# 翻译结果中能展示词典信息吗？

Bob 的翻译功能目前主要针对段落翻译，不过如果翻译服务返回了单词的词典结果，我也会一并展示。

目前主要有以下文本翻译服务支持显示词典结果：

* 有道翻译（需付费，但是注册赠送体验金，能用很久）
* 金山词霸（免费）
* 简明英汉词典增强版（免费，可离线使用）

可以在 [这篇文章](general/quickstart/service) 查看更多的其他服务。
